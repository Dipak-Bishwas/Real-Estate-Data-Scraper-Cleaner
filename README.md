# Real Estate Data Scraper & Cleaner

## Overview
Developed an automated real estate web scraping project using Python and Selenium to collect property listing data from websites. The project extracts raw property information, cleans and preprocesses the data using Pandas and NumPy, and converts it into a structured dataset for analysis and reporting.

This project helps automate manual data collection and improves efficiency in handling large volumes of real estate data.

---

## Key Features
- Automated property data extraction
- Dynamic webpage scraping using Selenium
- Data cleaning and preprocessing
- Removal of missing and duplicate values
- Structured Excel dataset generation
- Automated workflow for faster data collection

---

## Technologies Used
- Python
- Selenium
- Pandas
- NumPy
- OpenPyXL
- Jupyter Notebook

---

## Project Output
The project generates a cleaned and structured Excel dataset containing real estate property information for further analysis and visualization.
